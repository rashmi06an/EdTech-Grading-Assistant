# Automated EdTech Grading Assistant

An AI-powered handwritten digit classification system designed as the first step toward automating educational grading workflows. This project uses the MNIST dataset to train and evaluate machine learning models capable of recognizing handwritten digits (0–9) with high accuracy.

The project focuses on building an end-to-end pipeline covering data preprocessing, model training, hyperparameter tuning, performance evaluation, and an interactive web interface for real-time digit prediction.

## Tech Stack

- Python
- Scikit-learn
- NumPy
- Pandas
- Matplotlib
- HTML5, JavaScript
